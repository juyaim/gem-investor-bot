# 📈 GEM 投資判斷系統：全球事件與台灣供應鏈深度檢索模組

本工具旨在將「碎片化新聞」轉化為「具備第二層思考的供應鏈佈局地圖」。透過鎖定權威數據源，避開市場雜訊。

---

### 🎯 核心分析邏輯：三層思考法 (Triple-Layer Thinking)
1.  **第一層（市場直覺）：** 辨識大眾恐慌或樂觀的表面原因（如：股價跌不跌？服務會中斷嗎？）。
2.  **第二層（產業實相）：** 分析企業端真實痛點與補償機制（如：因攻擊而增加備援、因地緣政治而產能位移）。
3.  **第三層（供應鏈佈局）：** 精確定位受惠的台灣關鍵類股（散熱、先進封裝、網通、電源管理等）。

### 🔍 深度檢索指令示例 (Direct Commands)
* **輸入 `2330` / `6669`：** 自動檢索 [公開資訊觀測站](https://mops.twse.com.tw/mops/) 法說會簡報，分析資本支出與技術節點。
* **輸入 `川普關稅` / `中東衝突`：** 檢索 [台經院](https://tie.tier.org.tw/) 與 [資策會 MIC](https://mic.iii.org.tw/) 報告，分析產能轉單與非自願性提前備貨效應。
* **輸入 `通膨` / `降息`：** 連動 [財經 M 平方](https://www.macromicro.me/)，判斷全球資金流向與企業成本轉嫁能力。

### 📡 權威數據監控網址 (Authorized Data Sources)

#### 🏛️ 官方智庫與法說會 (時效與權威)
* [公開資訊觀測站 (MOPS)](https://mops.twse.com.tw/mops/) - 最原始的法說會簡報與電子書。
* [台灣經濟研究院 (TIER)](https://tie.tier.org.tw/db/article/index.aspx) - 深度產業結構與景氣預測。
* [資策會 MIC](https://mic.iii.org.tw/aisp/research-database) - 數位轉型與 ICT 產業權威報告。
* [工研院 IEKNet](https://ieknet.iek.org.tw/) - 先進技術與半導體趨勢分析。

#### 📊 財報與產業細節 (驗證與佈局)
* [財報狗 (Statementdog)](https://statementdog.com/) - 圖像化財報與毛利追蹤。
* [Goodinfo! 台灣股市資訊網](https://goodinfo.tw/tw/StockList.asp) - 類股連動與概念股清單查詢。
* [優分析 (UAnalyze)](https://uanalyze.com.tw/) - 供應鏈上下游關係拆解。
* [HiStock 嗨投資](https://histock.tw/) - 市場心理與券商觀點匯整。

#### 🌎 全球宏觀與資金流向 (大環境與循環)
* [財經 M 平方 (MacroMicro)](https://www.macromicro.me/macro/tw) - 核心宏觀數據與景氣循環位階。
* [StockQ 國際股市](https://www.stockq.org/market/) - 全球資產類別即時概覽。
