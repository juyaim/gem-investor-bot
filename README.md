GEM 投資判斷系統：全球事件與台灣供應鏈深度檢索

本工具旨在將「碎片化新聞」轉化為「具備第二層思考的供應鏈佈局地圖」。

### 🚀 核心分析邏輯：三層思考法 (Triple-Layer Thinking)
1. **[第一層：市場直覺]**：辨識大眾恐慌或樂觀的表面原因。
2. **[第二層：產業實相]**：分析企業真實痛點與補償機制（如：因地緣政治導致的產能位移）。
3. **[第三層：供應鏈佈局]**：精確定位受惠的台灣關鍵類股（散熱、先進封裝、電源管理等）。

### 🔍 深度檢索指令示例
* **輸入 4 位數股號 (如 2330 / 6669)**：自動檢索 [公開資訊觀測站] 法說會簡報。
* **輸入 宏觀關鍵字 (如 原油 / 通膨)**：自動連動 [財經 M 平方] 核心數據。
* **輸入 產業關鍵字 (如 CoWoS / 矽光子)**：啟動 AI 生成供應鏈連動圖譜。
